###第一步：创建做包工程

使用指令light create -t app创建做包工程。

###第二步：编辑菜单及配置文件

编辑native目录下的资源文件，如native/config.js定义整包配置项目。

第三步：打包及上传

使用light release -p指令做打包操作，将打包zip包上传到平台对应应用的条目内。

###第四步：在平台集成出安装包
