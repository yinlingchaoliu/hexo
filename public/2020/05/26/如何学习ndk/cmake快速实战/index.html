makefile前言

我写过makefile，编写过简单makefile框架,
但是cmake比较简单一些。
原理简单
编写输入：环境变量，cpp, c，h文件
编写输出：lib, so, bin (即静态库，动态库，二进制执行文件)

cmake官网

####常用方法

  • set 设置变量
  • include_directories 将头文件添加到搜索路径中
  • aux_source_directory 当前目录所有源文件保存到指定变量
  • add_subdirectory 添加子目录
  • add_library 指定要编译的库
  • add_executable 生成执行文件
  • target_link_libraries 将库链接
  • add_test 测试工具
